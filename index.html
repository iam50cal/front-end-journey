First a quick overview of the primary technologies you will learn and use!

HTML & CSS

HTML & CSS are two essential technologies you will be using in front-end development. HTML and CSS go hand in hand. HTML(Hypertext Markup Language) is a markup language you will use to structure your the content on your website/webapp. HTML uses tags to allow you to define your elements. These elements can be images, links, paragraphs, list and many more. The browser looks at these tags as instructions on how it should display content.

CSS(Cascading Style Sheets) is a stylesheet language that we use to define the layout and appeareance of our HTML elements. CSS allows us to apply styles to our HTML like changing colors, positioning, fonts and many other styles to our HTML elements. This is where things start to get fun early on in your journey as you'll start seeing your projects to look as you want. You will learn to use layout systems like Grid & Flexbox which allow you to position elemnents as you desire. I recomend expermenting and building plenty of layouts when you reach this point, these two click after consitent practice, at least that was my case.

This is just a very brief overview of these two technologies. There are plenty of resources online for you dive in and start learning building of these two technologies alone. Later on I will create a post listing a few resources I have used.
